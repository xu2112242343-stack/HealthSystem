## 使用说明

1. 在 VS Code / Cursor 中打开本文件，安装 **Mermaid** 预览插件即可渲染三张架构图。  
2. 导出为 PNG/SVG：可用 [Mermaid Live Editor](https://mermaid.live) 粘贴对应代码块导出。  
3. 若需与例图完全一致的**方框连线手绘风格**，可将各节「文字树」复制到 draw.io / ProcessOn 的「组织结构图」模板中快速排版。
