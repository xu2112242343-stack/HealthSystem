# Task: 编写项目设计思路文档

## Goal
为 HealthSystem 项目编写一份"设计思路"文档，参照用户提供的框架，使用 humanizer-zh skill 进行人性化中文润色。

## Scope
- 项目定位与目标场景
- 总体架构（四端 + 统一后端）
- 数据流设计（问卷→落库→评估→推荐→展示）
- 关键实现取舍
- 可扩展性设计

## Output
- 设计思路文档（Markdown），写入项目根目录或 docs/ 下

## Success Criteria
- 覆盖用户提供的所有主题点
- 语言自然、去AI味、符合中文技术写作习惯
- 引用项目实际代码结构作为佐证
- 使用 humanizer-zh 规则完成
